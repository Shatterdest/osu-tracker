# osu-tracker
A web server based osu! stats/session tracker

## Credits
- osu! API by osu!
- Score Inspector API by Amayakase
- Score Rank API by Respektive

## How to build from source:
- Windows
  - Visual Studio (Recommended)
- Linux
  - Use bash scripts
    - "build-release.sh"
    - "build-debug.sh"
